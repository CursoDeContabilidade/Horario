💡 **Dica para estudar contabilidade:**
Esse tipo de horário mostra algo importante: os cursos técnicos misturam **teoria + prática**. Por exemplo:

* Economia → entender o sistema financeiro
* Informática → usar programas de contabilidade
* Recursos Humanos → lidar com salários e impostos

Ou seja, contabilidade não é só números — envolve **gestão, economia e informática**.
